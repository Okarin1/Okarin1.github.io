## 步骤
我们可以在源文件 config.js 里面的 nav 标签下的联系的 items里面 添加如下代码：


```sh
{
    "text": "QQ",
    "link": "http://wpa.qq.com/msgrd?v=3&uin=填写你的QQ&site=qq&menu=yes",
    "icon": "reco-qq"
}
```


## 注意

在文字处填写QQ账号，注意与前一个用逗号隔开

## 跳转效果

[点击跳转](http://wpa.qq.com/msgrd?v=3&uin=201747922&site=qq&menu=yes)
